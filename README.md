# Marocain Name Generator 🇲🇦

A simple Python tool to generate Moroccan-style username combinations using names, years (2000–2026), and symbols.

---

## 🚀 Features

* Generate multiple variations of names
* Supports years from 2000 to 2026
* Includes different formats:

  * Lowercase (ziyad2000)
  * Capitalized (Ziyad2000)
  * With symbols (_ - @)

---

## 🛠️ Requirements

* Python 3

---

## ▶️ Usage

Run the script:

```bash
python3 generator.py
```

---

## 📂 Output

The script will generate a file:

```
marocain_names.list
```
